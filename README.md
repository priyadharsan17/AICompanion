# AICompanion

## AICodeCompanion

PySide6 + QML desktop app is available in `/AICodeCompanion` with:

- `Screens/Main.qml` for the sci-fi themed UI
- `Backend/companion_backend.py` for Python backend signals/slots

Run locally:

```bash
cd /home/runner/work/AICompanion/AICompanion/AICodeCompanion
python main.py
```
